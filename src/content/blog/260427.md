## 在 VPS 上创建 VPN 服务

1. VLESS + REALITY

安全性：采用消除特征的 REALITY 传输方式，让你的 VPN 流量看起来完全像是访问真实的著名网站，如微软、谷歌、雅虎等的 HTTPS 流量。

优点：无需购买域名或配置 TLS 证书，极难被防火墙通过指纹识别发现，且速度极快。

2. Hysteria2

安全性：基于 QUIC 协议（UDP），模拟现代网页浏览行为。

优点：在丢包严重的恶劣网络环境下，如高峰期，表现极佳，能大幅提升下载和加载速度。

#### VLESS + REALITY

1. 放行防火墙

```bash
iptables -I INPUT -p tcp --dport 443 -j ACCEPT
```

2. 安装脚本

```bash
wget --no-check-certificate -O ${HOME}/Xray-script.sh https://raw.githubusercontent.com/zxcvos/Xray-script/main/install.sh && bash ${HOME}/Xray-script.sh
```

3. 复制 VLESS

复制 VLESS，使用代理工具：https://github.com/2dust/v2rayN/releases
